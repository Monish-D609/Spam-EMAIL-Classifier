# Email Spam Classifier

A machine learning based email spam classifier using
TF-IDF and Logistic Regression with threshold tuning.

## Tech Stack
- Python
- Scikit-learn
- Streamlit

## How It Works
1. User enters email text
2. Text is vectorized using TF-IDF
3. Model predicts spam probability
4. Custom threshold determines Spam / Ham

## Deployment
Deployed using Streamlit Cloud via GitHub.
